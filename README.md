# Ordox

🌏 **Ordox** is a decentralized voice and messaging platform designed for control, low-latency communication, and self-hosted infrastructure.

> Stay connected.

---

## About

Ordox is an alternative to centralized communication platforms. It gives users full control over their network, data, and communication flow.

The project focuses on:
- self-hosted communication
- low-latency voice transmission
- decentralized architecture
